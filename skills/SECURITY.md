# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest on `main` | Yes |
| Older commits | No |

## Reporting a Vulnerability

If you discover a security vulnerability in GHS, please report it responsibly.

**Do NOT open a public GitHub issue for security vulnerabilities.**

### How to Report

1. Email the maintainers via the [Atypical Consulting](https://github.com/Atypical-Consulting) organization contact
2. Alternatively, use [GitHub's private vulnerability reporting](https://github.com/Atypical-Consulting/GitHubSkills/security/advisories/new)

### What to Include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

### What to Expect

- **Acknowledgment** within 48 hours
- **Status update** within 7 days
- **Resolution or mitigation plan** within 30 days

We appreciate your help in keeping GHS and its users safe.
